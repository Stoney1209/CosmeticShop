Optimiza el siguiente prompt aplicando las mejores prácticas de engineering de Anthropic:

**Prompt original:**
$ARGUMENTS

**Directrices para optimizar:**
1. **Sé explícito**: Specifies exactly what result you want, don't rely on vague inferences
2. **Añade contexto**: Explain the purpose or motivation behind the task so the model understands better
3. **Specify format**: Indicate clearly the desired output format (prose, bullet points, code, JSON, etc.)
4. **Control verbosity**: Indicate if you want detailed or concise answers
5. **Be specific about behavior**: Define exactly how the model should behave (action vs suggestion, autonomy level, etc.)
6. **Usa ejemplos**: If there's a specific format you want, show an example

**Tu respuesta debe incluir:**
1. El prompt optimizado claramente labelado
2. Una breve explicación de los cambios realizados y por qué mejoran el prompt
